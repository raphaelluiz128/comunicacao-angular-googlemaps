# Coords Test API

## Setup

```bash
# install adonis CLI
npm i -g @adonisjs/cli

# install dependencies
npm install

# create env file
cp .env.example .env

# generate app key
adonis key:generate

# edit configs
nano .env

# run migrations
adonis migration:run

# run seeds
adonis seed

# start server
adonis serve --dev
```
